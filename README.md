# Student Record Management System

This project is a Student Record Management System designed to manage student data efficiently. It supports adding, editing, viewing, and deleting student records. Below are the instructions to deploy the application on Vercel.

## Table of Contents
- [Requirements](#requirements)
- [Installation](#installation)
- [Deployment Instructions](#deployment-instructions)
- [Usage](#usage)

## Requirements
- Node.js (version 14 or later)
- MongoDB (for database storage)
- Vercel account

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Harit2552/Student-Record-Managment-System.git
   cd Student-Record-Managment-System
   ```
2. Install the required packages:
   ```bash
   npm install
   ```

## Deployment Instructions
To deploy this application on Vercel, follow these steps:

1. **Create a Vercel Account:** If you don't have a Vercel account, sign up at [Vercel](https://vercel.com/).

2. **Import GitHub Repository:**
   - Go to the Vercel dashboard.
   - Click on the "New Project" button.
   - Select GitHub and authorize Vercel to access your repositories.
   - Choose the `Student-Record-Managment-System` repository from the list.

3. **Configure the Deployment:**
   - Set the necessary environment variables in Vercel. You may need to add your MongoDB connection string.
   - Configure build settings if necessary (default settings should work).

4. **Deploy: ** Click on the "Deploy" button to start the deployment process.

5. **Access Your Application:** Once the deployment is complete, Vercel will provide a URL where your application is hosted.

## Usage
To use the Student Record Management System, navigate to the deployed URL and you will be able to:
- Add new student records.
- Edit existing records.
- View all student records.
- Delete records.
